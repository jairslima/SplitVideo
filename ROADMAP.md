# Roadmap

## Melhorias sugeridas

### Interface grafica

Uma GUI simples com selecao de arquivo, lista de cortes e barra de progresso reduziria o atrito para uso recorrente.

### Drag and drop

Aceitar arrastar um arquivo para o `.exe` ou para a janela da futura GUI melhoraria o fluxo no Windows.

### Perfil de nomes

Ja existe `--name-mode short`, mas ainda vale adicionar modelos extras de nomenclatura e presets por projeto.

### Log persistente

Ja existe log JSON por execucao. O proximo passo util e exportar tambem em `.txt` para leitura rapida.

### Validacao por streams

Ja existe `--verify`, mas ainda vale acrescentar tolerancias configuraveis e comparacao mais rica de streams.

### Corte mais preciso

Oferecer um modo opcional com recodificacao parcial ou total para quem quiser mais precisao de frame.
